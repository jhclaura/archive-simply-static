AHHH why is it so difficult to export rigged model to Three.js?<\/p>\n

This is a reminder post of how did I export with correct joint rotations.<\/p>\n

 <\/p>\n

[From Maya] – export selection as FBX, set Up Axis from Y to Z<\/p>\n

<\/p>\n

 <\/p>\n

[Import to Blender] – Scale up!<\/p>\n

 <\/p>\n

[Export to Three.js] – Select Mesh and export as Three.js. Geometry: also select “Bones” and “Skinning”. Type: choose “Geometry” instead of BufferGeometry. Skeletal animations: select “Rest” instead of “Pose”<\/p>\n

<\/p>\n

 <\/p>\n

[Import to Three.js] – Load with THREE.JSONLoader. Set “skinning: true” in material. Create with\u00a0THREE.SkinnedMesh. And finally manipulate with\u00a0THREE.SkinnedMesh’s skeleton.bones array.<\/p>\n
