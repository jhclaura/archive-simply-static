A experiment of data viz in virtual reality in The Economist Media Lab, with world famous Big Mac Index. It has two parts – motion graphics style intro, and interactive map viewing.

Two data sets are each represented by burger tower and map color. Through different time periods, the height of the burger tower and the color of the map change.

It’s made with Unity3D and dynamically generated with JSON file. Read more here.
